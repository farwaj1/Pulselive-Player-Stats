# Pulselive-statsCard

This is a stats card component where the user is able to select from a list of players to view the relevant stats for that player. The stats are stored inside a table so its reusable and its a more efficient way to add/remove stats. 

## Table of Contents

- [Installation](#installation)
- [Run Project](#run-project)


## Installation 

Open terminal and run command: 

```
npm install http-server -g

```

This command uses NPM to install the http-server

In the root folder of your project run this command:


```
npm install

```

Now we need to install package node-sass:

```
npm run scss

```

This looks out for the scss files in the scss folder and re-compiles them when the scss file is updated. 

## Run Project

To run this project, open up terminal in the root folder of your project and run this command:

```
http-server

```
This command will start the server and you will be given a link to open and view in your browser:
http://192.168.0.43:8080/app/
